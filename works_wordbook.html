単語帳アプリケーション

概要

日本語と英語の単語帳アプリケーションです。
和訳と英訳の問題作成、クイズができます。
スマートフォンからも実行可能です。

目的

以下の3点から単語帳アプリケーションを作成することにしました。

  • PHPを使用してログイン機能、CRUD機能があるwebアプリケーションの作成
  • クリック（タップ）のみで簡単に何度も繰り返し遊べること
  • 遊んでるうちに多少なりとも知識が身につくこと

使用言語

  • PHP
  • JavaScript(jQueryも含む)
  • HTML
  • CSS
  • MySQL

開発環境

  • Windows10
  • XAMPP
  • Visual Studio Code
  • git(SourceTree使用)

機能一覧

  • トップ画面
  • ログイン画面
    • ログイン処理
    • ユーザー登録
  • 問題編集画面
    • 登録
    • 更新
    • 削除
    • データ一覧表示
  • クイズ画面
    • 出題設定
    • クイズ解答
    • 結果表示
